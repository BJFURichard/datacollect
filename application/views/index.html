系统简介

移动客户端

本系统的移动端为一个安卓APP，在护林员巡护过程中，打开客户端便可自动向系统发送位置信息。

功能模块

移动客户端除了发送位置信息以外，还可以进行树高测量和角规绕测。其中，角规绕测模块使用前需要进行完成校准。

使用的主要技术

在Web中，主要使用到了Amaze UI作为网站的前端开发框架，地图展示使用了百度地图的API。手机客户端基于Android开发，主要使用到的硬件设备有GPS模块、摄像头、加速度传感器，在角规成像时使用了OpenCV进行自动识别。

数据接口

本系统所有的数据交互统一采用 JSON 格式。数据的接口暂时说明不对外公布，数据统一使用JSEND规范格式，如需了解JSEND请点击这里。
